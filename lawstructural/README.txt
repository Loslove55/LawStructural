README for "Information Asymmetry in Profit-generating Graduate Education 
Markets: A Structural Approach to Law Schools" by Philip J. Erickson
==========================================================================

What is this?
-------------

Program for performing the estimation and simulation detailed in the above paper.


What is here?
-------------

Code for the paper, data files, nose tests

Dependencies
------------

Python:
1. NumPy (http://www.numpy.org/)
2. Pandas (http://pandas.pydata.org/)
3. KNITRO (http://www.ziena.com/knitro.htm)
4. RPy2
5. MatPlotLib
6. distutils
7. distribute
8. nose
9. virtualenv

R:
1. mice
2. VGAM
3. texreg
4. xtable

C:
1. lapacke
2. cblas

Note: NumPy, Pandas, and MatPlotLib are included in Enthought Canopy 
      (https://www.enthought.com/products/canopy/) which, as of 
      this version, is free for academics. All the required python packages
      are included in the Anaconda python distribution, which can be
      downloaded from http://continuum.io
